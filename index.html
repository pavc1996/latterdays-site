LATTERDAYS

Five directions for the landing page — each one built on a different structural idea, not just a different coat of paint. Click through, pick a favorite, the rest get cut.

01 / The Issue
Editorial — Scroll-Snap Plates
"a hand-bound issue, one plate at a time"

Structure: vertical scroll-snap spreads on black. Each garment is a numbered plate, not a grid tile. Spine + dot-index instead of a nav bar. Reads like a quiet fashion monograph.

View → 02 / A Walk Through the Pond
Painted — Parallax Narrative
"one long letter, written over the painting"

Structure: pinned Sabr painting as backdrop; verses reveal as you scroll. Archive pieces appear as painted memory clusters, not a gallery. Finishes on a wax-sealed notify letter.

View → 03 / Course Catalog
Ivy — Division of Patient Studies
"enroll in the drop"

Structure: the site is a university catalog. Left sidebar of courses (LTD 101 Bloom · 201 Arc · 301 Sabr), right panel shows the selected course with garments as roster ID-cards. Notify = enrollment application.

View → 04 / Issue 03
Zine — Horizontal Page-Turn
"flip through it like a magazine"

Structure: horizontal scroll-snap pages with a page counter. Cover → ToC → editor's letter → Sabr portfolio → Arc lookbook → LTDZ field notes → Bloom archive → classifieds (notify) → back cover.

View → 05 / The Wall
Brutalist — Bulletin Board
"posters, ticker, countdown"

Structure: marquee ticker + live countdown, a scattered taped-poster wall of every drop image, doctrine blocks, archive styled as police-evidence list, notify as a demand. Corteiz/Balenciaga bulletin energy.
